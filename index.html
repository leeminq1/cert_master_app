개인정보처리방침

최종 업데이트: 2025년 5월 4일

🔒 이 앱은 어떤 개인정보도 수집하거나 외부 서버로 전송하지 않습니다. 모든 데이터는 사용자의 기기 내에만 저장됩니다.

1. 수집하는 정보

자격증 마스터는 개인 식별 정보를 수집하지 않습니다. 회원가입, 로그인, 이메일 주소, 이름, 위치 정보 등 어떠한 개인정보도 요구하지 않습니다.

2. 기기 내 저장 데이터

앱의 정상적인 작동을 위해 다음 데이터를 사용자의 기기 로컬 저장소에만 저장합니다:

  • 학습 진도 및 복습 일정 (간격 반복 알고리즘 데이터)
  • 문제 풀이 기록 및 정답률
  • 북마크된 문항 목록
  • 앱 설정 (테마, 알림 시간, 학습 목표 등)
  • 일별 학습 활동 기록 (히트맵 표시용)

이 데이터는 오직 해당 기기에만 존재하며, 외부로 전송되지 않습니다.

3. 인터넷 권한

앱은 인터넷 연결 권한을 요청하지 않습니다. 오프라인 환경에서 완전히 동작합니다.

4. 제3자 서비스

자격증 마스터는 광고 네트워크, 분석 서비스, 소셜 미디어 플랫폼 등 어떠한 제3자 SDK도 사용하지 않습니다.

데이터 내보내기(백업) 기능 사용 시, 사용자가 직접 선택한 공유 앱(이메일, 클라우드 스토리지 등)을 통해 파일이 전송됩니다. 이 과정은 사용자가 직접 제어하며, 앱이 자동으로 전송하지 않습니다.

5. 알림 권한

매일 학습 알림을 원하는 경우 알림 권한을 요청합니다. 이 권한은 기기 내에서 예약된 알림 표시에만 사용되며, 어떤 데이터도 외부로 전송하지 않습니다. 알림은 언제든지 기기 설정에서 비활성화할 수 있습니다.

6. 아동 개인정보

본 앱은 만 13세 미만 아동으로부터 의도적으로 정보를 수집하지 않습니다. 앱이 개인정보를 수집하지 않으므로 아동 이용에도 안전합니다.

7. 방침 변경

개인정보처리방침이 변경될 경우, 이 페이지에 최신 내용을 게시합니다. 변경 전에 본 페이지를 정기적으로 확인하시기를 권장합니다.
